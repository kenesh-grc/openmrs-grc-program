This repository documents a hands-on Governance, Risk, and Compliance (GRC)

## Objectives
- Perform a healthcare-focused risk assessment

- Document policies, procedures, and governance artifacts
- Demonstrate executive-level risk reporting

## Frameworks & Standards
- NIST 800-53
- NIST Risk Management Framework (RMF)
- HIPAA Security Rule (contextual)

## Scope
- OpenMRS electronic medical record system
- AWS cloud hosting environment
- Protected Health Information (PHI)

This lab simulates real-world GRC analyst responsibilities.
